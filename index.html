What is Sharkaby?

Tiling Windows

Sharkaby focuses fully on minimal tiling WMs, so we can give you the best experience for advanced users.

Solid Software

These wms comes with your favorite programs like Alacritty, Polybar and NvChad.

Minimal Base

Built with Lua, V and Smalltalk, Sharkaby follows the KISS principles, keeping the base system minimal yet flexible.
